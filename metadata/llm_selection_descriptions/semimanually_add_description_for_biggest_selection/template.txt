This dataset focuses on the production of selected goods and services in the structure published by Eurostat (PRODCOM data). The data is collected at the "Roky" (Year) level and covers the "Území" (Territory) type "State". The dataset includes the following columns: "Území", "Roky", "Seznam výrobků", and "Ukazatel".

For "Území", the available value is:
- "Česko"

For "Roky", the available value is:
- "2023"

For "Seznam výrobků", the available values are:


For "Ukazatel", the available values are:
- "Objem produkce - celková produkce"
- "Objem produkce - prodaná produkce celkem"
- "Objem produkce - prodaná produkce vlastní a průmyslové služby"
- "Objem produkce - prodaná produkce v režimu práce ve mzdě"
- "Hodnota produkce v tis. Kč - prodaná produkce celkem"
- "Hodnota produkce v tis. Kč - prodaná produkce vlastní a průmyslové služby"
- "Hodnota produkce v tis. Kč - Prodaná produkce v režimu práce ve mzdě"

The data is organized in a flat structure, where each record is uniquely identified by the combination of "Území", "Roky", "Seznam výrobků", and "Ukazatel". There is no explicit hierarchy among the columns; each combination represents a specific measurement of a product or service for a given year and territory, with a specific indicator.

This structure allows for detailed semantic search and vector embedding applications, as each dimension and its possible values are explicitly defined and exhaustive.
